Component Architecture: ChessBoard
ChessBoard.tsx
1 Receive gameState prop → convert to FEN
2 Render react-chessboard with position, orientation, styles
3 Handle onPieceDrop → validate via chess.js → call onMove
4 Apply custom square styles for legal moves, last move, check
Props & Callbacks
react-chessboard
A Renders 8×8 SVG board with pieces
B Handles drag-and-drop animations
C Applies custom square highlights
✓ VALID MOVE
Call onMove(move) → parent updates gameState
✗ INVALID MOVE
Return false → piece snaps back
