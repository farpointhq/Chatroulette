Move Handling Flow
USER INTERACTION
1 User clicks or drags a piece from square A
2 Component generates legal moves for that square
3 Legal destination squares highlighted in green
Drop / Click destination
VALIDATION & EXECUTION
4 Check if destination is in legal moves list
5 If pawn promotion needed → show picker overlay
6 Construct Move object with from, to, promotion?
7 Call onMove(move) → parent updates game state
✓ VALID
Move accepted, board updates, last-move highlighted
✗ INVALID
Piece snaps back, no state change
