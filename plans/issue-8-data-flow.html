Component Data Flow
APP (State Container)
1 useState manages: status, nickname, onlinePlayers, matchFound, errorMessage
2 Callbacks: onFindMatch, onCancelSearch, onAcceptMatch, onDeclineMatch, onRetry
Props down, Events up
LOBBY COMPONENT
1 Receives all props from App
2 Switches on status to render sub-component:
• idle → LandingPage (nickname, online count, Find Match)
• searching → MatchmakingQueue (spinner, cancel, wait time)
• match-found → OpponentFound (opponent, color, countdown, accept/decline)
• error → ConnectionError overlay (message, retry)
