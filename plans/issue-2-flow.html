SERVER STARTUP
1Load .env → PORT, CORS_ORIGIN
2Create Express app + HTTP server
3Attach Socket.io with CORS config
4Register GET /health endpoint
5Listen on PORT
Client Connects
CONNECTION LIFECYCLE
1io.on('connection') fires
2Socket auto-joins 'lobby' namespace
3Emit 'room:list' with active rooms
4Wait for events: 'room:create', 'room:join', 'room:leave'
CREATE ROOM
Generate roomId → create RoomState → socket.join(roomId) → broadcast 'room:created'
JOIN ROOM
Validate room exists → socket.join(roomId) → update RoomState → broadcast 'room:updated'
DISCONNECT
1socket.on('disconnect') fires
2Remove socket from all rooms
3If room empty → delete room
4Broadcast updated room list
