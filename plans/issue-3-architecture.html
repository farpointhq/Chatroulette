Chess Engine Architecture
UI LAYER
1 ChessBoard.tsx — React component, 8×8 grid, drag-and-drop, promotion picker
2 Game Hook — Manages game state, calls engine for validation
API calls
ENGINE LAYER
1 ChessEngine (chess.js wrapper) — Main entry point: .move(), .fen(), .pgn()
2 game.ts — createGame(), makeMove(), undoMove(), getGameResult()
3 moves.ts — generateLegalMoves(), isInCheck(), isSquareAttacked()
4 special.ts — Castling, en passant, promotion validation
5 fen.ts — FEN parse/generate, STARTING_FEN
6 pgn.ts — SAN notation, generatePGN() with headers
7 board.ts — Board primitives: coordinates, piece get/set, board creation
delegates to
CORE LIBRARY
★ chess.js — Battle-tested library handling all rule logic, move generation, check detection
